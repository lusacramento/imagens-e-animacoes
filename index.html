Teoria

Problemas

  • Imagens muito grandes (largura e altura);
  • Imagens muito pesadas (excesso de KB);
  • Imagens não nítidas com baixíssima resolução;
  • Formato da imagem inadequado (cada formato tem seu propósito);
  • Imagens “quebradas” (caminho/URL incorreto);
  • Qualidade de compressão x Peso.

Vantagens de Otimização

  • Deixa o carregamento do site mais rápido;
  • Aprimora a experiência de quem usa o site/aplicativo;
  • Melhora as chances de o site ficar em melhores posições no Google;
  • Diminui o número de abandonos do site (que acontecem porque a página demorou demais para carregar);
  • Aumenta o número de páginas do site visualizadas em cada sessão;
  • Impulsiona conversões;
  • Melhora suas chances de se sobressair ante projetos concorrentes.

Formatos de Imagem

JPG

  • Compressão com perda;
  • Imagens com peso mais leves;
  • Não suporta imagens transparentes (sem canal alfa).

Ideal para fotgrafias e representações de cenas realistas.

PNG

  • Compressão com perda;
  • Imagens com peso maior que JPG;
  • Suporta transparência (canal alpha);
  • Existem duas versões:
    • PNG-8 - suporta apenas 256 cores (8 bits). Ideal para logos, ilustrações, gráficos, ícones e backgrounds;
    • PNG-24 - suporta 16.777.216 cores. PNG-24: Ideal para composição estética com sobreposição de camadas (devido sua transparência).

GIF

  • Suporta animações;
  • Baixa qualidade de imagem;
  • Suporta somente 256 cores.

Ideal para animações de tamanho pequeno e duração curta.

WebP

  • Formato criado pela Google;
  • Imagens com tamanho menor comparado a JPG e PNG;
  • Suporta transparência (canal alpha);
  • Não é suportado por alguns navegadores.

Recomendado para todo tipo de imagem estática, porém, ainda é pouco utilizado.

SVG

  • Imagem vetorial. Permite redimensionar para qualquer tamanho;
  • Peso mais leve;
  • Requer experiência em design gráfico para vetorizar a imagem.

Recomendado para gráficos.

Otimização de Imagens

Edição de imagens

Precisamos adequar o tamanho da imagem e sua resolução. Este tipo de tratamento pode ser feito de várias maneiras:

  • Com editor de imagens profissionais, como por exemplo:
    • Adobe FotoShop, Adobe LiteRoom e Gimp (Linux) para fotografias;
    • Adobe Illustrator e InkScape (Linux) para imagens e gráficos vetoriais.
  • Serviços on-line como por exemplo:
    • TinyPNG/TinyJPG
    • JPEG Mini
  • Plugins e conversores de imagem como por exemplo:
    • Plugins de Wordpress como:
      • Smush
      • Imagify
    • Ferramentas de linha de comando como o pacote ImageMagick (Linux).

Dicas para otimização de Imagens

  • Procure definir previamente qual o público alvo e em qual dispositivo ele irá acessar seu conteúdo;
  • Use preferencialmente os tamanhos:
    • 150 x 150 pixels - Miniatura, como ícones e pequenas logomarcas
    • 300 x 300 pixels - Pequena, para imagens em geral;
    • 600 x 600 pixels - Média, para imagens em geral;
    • 1024 x 1024 pixels - Grande, para cabeçalhos e imagens em background;
    • Até 1900 pixels - Full Screen, ideal para imagens em tela cheia para telas grandes, como TV por exemplo;
  • Utilize uma resolução das imagens em 72 PPI (pixels por polegadas);
  • Não deixe os tamanhos dos arquivos da imagens ultrapassar 150KB para imagens médias e 400KB em Full Screen;
  • Caso haja necessidade em ter imagens com maior resolução e tamanhos em seu projeto, opte por gerar duas imagens, apresente a imagem pequena por padrão e deixe o acesso a imagem grande em outra página ou disponibilize-a para download como exemplo.

Tags e atributos para imagem.

Utilização das tags <img>, <figure> e <figcaption> para criar imagens.

Propriedades:
  • src - caminho da imagem;
  • alt - descrição da imagem;
  • width - largura da imagem;
  • height - altura da imagem;
  • class - classe da imagem;
  • id - identificador da imagem;
  • title - título da imagem;

exemplo 1: 

index.html
<figure>
    <img 
        id="img-senac" class="img"
        style="border: solid 2px var(--color-yellow);
        border-radius: 10%;border;" 
        src="src/imgs/logo-senac-minas.webp" 
        alt="logotipo do Senac nas cores azul 
        e amarelo, em azul, no meio escrito 
        Senac e embaixo escrito Minas Gerais"
        width="auto"
        height="auto"
        title="Logo Senac MG" 
    />
    <figcaption>
        Legenda: logotipo Senac Minas
    </figcaption>
</figure>

Animando Elementos com CSS

Propriedades de classe animation

Utilização da proprieadade <animation> e suas sub-propriedades:

  • animation-delay - atraso da animação;
  • animation-direction - direção da animação;
  • animation-duration - duração da animação;
  • animation-iteration-count - número de repetições;
  • animation-name - nome da animação;
  • animation-play-state - estado de reprodução;
  • animation-timing-function - função de tempo;
  • animation-fill-mode - modo de preenchimento.

Utilização de @keyframes:

  • keyframes nome da animação;
  • from - Determina os atributos do estado inicial da animação;
  • to - Determina os atributos do estado final da animação.

Exemplo 2: 

index.html
<div class="animated-container">
    <h2 class="animation">Exemplo</h2>
  </div>

    
style.css
.animated-container {
    overflow: hidden; /* necessário para transbordamento */
  }
  
  .animation {
    animation-delay: 2s;
    animation-iteration-count: infinite;
    animation-duration: 5s;
    animation-name: slide;
    animation-direction: alternate; /* alterna entre direções */
    animation-play-state: running; /* estado inicial da animação */
  }
  
  .animation:hover {
    animation-play-state: paused; /* pausa a animação quando passar o mouse */
  }

  @keyframes slide { /* nome da animação */
    from {
      margin-left: 100%;
      color: var(--color-blue);
    }
  
    to {
      margin-left: 0%;
      font-size: 2rem; /* cuidado com redimensionamentos! */
      color: var(--color-yellow);
    }

Animando com a biblioteca Animatecss

Uma maneira fácil e rápida de criar animações é utilizando a biblioteca de estilosAnimate.css. Esta biblioteca já contém várias animações pré-configuradas.

Passos para utilização do Animate.css

  1. Importe a arquivo de estilo no cabeçaho do documento;
  2. Adicione a classe animate__animated animate__nome-da-animação, onde o nome-da-animação. encontra-se no site da biblioteca

Exemplo 3 

  index.html
  <h1 
    class="animated__animate anitmate__pulse animate__infinite">
</h1>

Animate

Observações

  • Utilize animações com parciomônia! Não exagere na quantidade!
  • Utilize apenas em elementos que deseja chamar a atenção!
  • Algumas propriedades, especialmente relativo ao tamanho width e height podem causar efeitos indesejáveis!
  • Cuidado com repetições infinitas. Tente evitar esse procedimento!
