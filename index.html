“Cria gráficos, imagens e animações otimizadas para a interface do website, seguindo princípios de comunicação visual coerentes com o projeto".

Teoria

Problemas

  • Imagens muito grandes (largura e altura);
  • Imagens muito pesadas (excesso de KB);
  • Formato da imagem inadequado (cada formato tem seu propósito);
  • Imagens “quebradas” (caminho/URL incorreto);
  • Qualidade de compressão x Peso.

Vantagens de Otimização

  • Deixa o carregamento do site mais rápido;
  • Aprimora a experiência de quem usa o site/aplicativo;
  • Melhora as chances de o site ficar em melhores posições no Google;
  • Diminui o número de abandonos do site (que acontecem porque a página demorou demais para carregar);
  • Aumenta o número de páginas do site visualizadas em cada sessão;
  • Impulsiona conversões;
  • Melhora suas chances de se sobressair ante projetos concorrentes.

Formatos de Imagem

JPG

  • Compressão com perda;
  • Imagens com peso mais leves;
  • Não suporta imagens transparentes (sem canal alfa).

Ideal para fotgrafias e representações de cenas realistas.

PNG

  • Compressão com perda;
  • Imagens com peso mais pesado que JPG;
  • Suporta transparência (canal alpha);
  • PNG-8 - suporta apenas 256 cores (8 bits);
  • PNG-24 - suporta 16.777.216 cores.

PNG-8: Ideal para logos, ilustrações, gráficos, ícones e backgrounds;
PNG-24: Ideal para composição estética com sobreposição de camadas (devido sua transparência).

GIF

  • Suporta animações;
  • Baixa qualidade de imagem;
  • Suporta somente 256 cores.
Ideal para animações de tamanho pequeno e duração curta.

WebP

  • Formato criado pela Google;
  • Imagens com tamanho menor comparado a JPG e PNG;
  • Suporta transparência (canal alpha);
  • Não é suportado por alguns navegadores.

Recomendado para todo tipo de imagem estática, porém, ainda é pouco utilizado.

SVG

  • Imagem vetorial. Permite redimensionar qualquer tamanho;
  • Peso mais leve;
  • Requer experiência em design gráfico para vetorizar a imagem.

    • Recomendado para gráficos

Tags e atributos para imagem.

Utilização da tag e <img> <figure> e <figcaption> para criar imagens. Propriedades:
  • src - caminho da imagem;
  • alt - descrição da imagem;
  • width - largura da imagem;
  • height - altura da imagem;
  • class - classe da imagem;
  • id - identificador da imagem;
  • title - título da imagem;
exemplo: 
<figure>
    <img 
        id="img-senac" class="img" 
        src="src/imgs/logo-senac-minas.webp" 
        alt="logotipo do Senac nas cores azul 
        e amarelo, em azul, no meio escrito 
        Senac e embaixo escrito Minas Gerais"
        width="auto"
        height="auto"
        title="Logo Senac MG" 
    />
    <figcaption>
        Legenda: logotipo Senac Minas
    </figcaption>
</figure>

Animando Elementos com CSS

Utilização da proprieadade <animation> e suas sub-propriedades:
  • animation-delay - atraso da animação;
  • animation-direction - direção da animação;
  • animation-duration - duração da animação;
  • animation-iteration-count - número de repetições;
  • animation-name - nome da animação;
  • animation-play-state - estado de reprodução;
  • animation-timing-function - função de tempo;
  • animation-fill-mode - modo de preenchimento.

Utilização de @keyframes:

  • keyframes nome da animação;
  • from - Determina os atributos do estado inicial da animação;
  • to - Determina os atributos do estado final da animação.
index.html

  <div class="animated-container">
    <h2 class="animation">Exemplo</h2>
  </div>

    
style.css
.animated-container {
    overflow: hidden; /* necessário para transbordamento */
  }
  
  .animation {
    animation-delay: 2s;
    animation-iteration-count: infinite;
    animation-duration: 5s;
    animation-name: slide;
    animation-direction: alternate; /* alterna entre direções */
    animation-play-state: running; /* estado inicial da animação */
  }
  
  .animation:hover {
    animation-play-state: paused; /* pausa a animação quando passar o mouse */
  }

  @keyframes slide { /* nome da animação */
    from {
      margin-left: 100%;
      color: var(--color-blue);
    }
  
    to {
      margin-left: 0%;
      font-size: 3rem;
      color: var(--color-yellow);
    }
